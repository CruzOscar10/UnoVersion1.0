# Juego UNO en Java

Este es un proyecto sencillo del juego UNO hecho en Java.

El programa simula una partida entre un jugador y la computadora usando solo cartas numéricas.

---

## ¿Qué hace el programa?

- Crea un mazo automáticamente
- Reparte cartas
- Controla los turnos
- Permite robar carta si no tienes una válida
- Detecta cuando alguien gana

El objetivo es quedarse sin cartas primero.

---

## Clases del proyecto

- Carta → Representa una carta (color y número)
- Mazo → Contiene todas las cartas
- Mano → Guarda las cartas de cada jugador
- Juego → Controla la lógica del juego
- Principal → Inicia el programa

---

## Cómo ejecutarlo

1. Abrir el proyecto en tu IDE.
2. Ejecutar la clase `Principal`.
3. Jugar desde la consola.

---

## Requisitos

Java 8 o superior.
